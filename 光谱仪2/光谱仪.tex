\subsection{光谱仪的组成}
一个基本的光谱仪包含以下部件：
\begin{itemize}
    \item 入射狭缝：控制入射光束宽度
    \item 准直系统：将发散光束准直
    \item 分光元件：三棱镜
    \item 聚焦系统：将分光后的光聚焦
    \item 观测屏/探测器：用于观测光谱
\end{itemize}
